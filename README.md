Простой веб-приложение с современным дизайном и удобным интерфейсом.

## Возможности

- Современный дизайн
- Адаптивная верстка
- Быстрая загрузка
- Поддержка мобильных устройств

## Установка

1. Клонируй репозиторий:
```bash
git clone https://github.com/botir07/webapp.git
```

2. Открой файл `index.html` в браузере

## Использование

Просто открой файл `index.html` в любом современном браузере.

## Структура проекта

```
webapp/
├── index.html       # Главный файл приложения
└── README.md        # Документация
```

## Разработка

Для разработки можно использовать любой текстовый редактор. Приложение написано на чистом HTML/CSS/JavaScript.
